# Assets Folder

Please place your Orbit Labs logo file here as `logo.png`.

The logo should be:
- PNG format (or SVG)
- Transparent background preferred
- Recommended size: 80x80px or larger (will be scaled down to 40px height)

If your logo file has a different name or format, update the `src` attribute in `index.html`:
- Line ~18: `<img src="assets/logo.png" alt="Orbit Labs Logo" class="logo-img">`
- Line ~350: `<img src="assets/logo.png" alt="Orbit Labs Logo" class="logo-img">`
